Reverse Engineering: Nipro Surdial 55Plus Fluidic Maintenance Components

1. Project Overview
	This repository contains the complete 3D mechanical CAD models, print-ready parametric STL files, and optimized 3MF slicing configuration profiles **for both Bybass coubling connector and Coupler Holder**. Additionally, it includes dedicated documentation featuring live photographs and video demonstrations of both manufactured parts undergoing strict verification and functional testing on the actual dialysis machine.
	This project was initiated in response to an urgent requirement from the **Ministry of Health (MOH)** due to the critical unavailability and loss of original OEM spare parts in local inventories. By employing precise reverse engineering methodologies, authentic components were sourced, measured, and successfully replicated to eliminate machine downtime and sustain uninterrupted patient care.

2. Targeted Equipment & Component Identification

	Device Manufacturer: Nipro Corporation
	Equipment Model:** Surdial™ 55Plus Hemodialysis System
	Component Descriptions:
  		1.Bypass Coupling Connector:Serves as a fluidic interconnector bridging the dialysate coupling lines (Red/Arterial & Blue/Venous) to complete a leak-proof closed-loop pathway required during automated rinsing, chemical cleaning, and disinfection cycles without a dialyzer attached.
  		2.Coupler Holder:A custom mechanical docking base mounted directly onto the machine chassis, engineered to firmly retain, support, and protect the coupling assembly when engaged in bypass mode or during clinical storage. It is precisely designed to engage and secure the Bypass Coupling Connector in a tight fit, maintaining the necessary tactile pressure on the internal switch/sensor to allow the machine to operate smoothly without triggering bypass errors.

3. Manufacturing Specifications

	Both components were designed and produced using the following practical specifications to ensure proper operation:

		1. CAD Software:Designed using FreeCAD.
		2. Material:ABS filament - Chosen for its strength, heat resistance, and ability to handle cleaning chemicals.
		3. Slicing Profiles:All specific printing parameters (such as infill and wall settings) are fully configured and ready inside the provided **3MF** files for direct printing.

4. Repository Structure & Live Testing

	The files and experimental assets within this repository are structured as follows:

		1. CAD&STL/: Contains the primary FreeCAD source design files and the parametric meshes for both components.

		2. Printingsetting/: Contains optimized slicing settings and configuration files ready for production.

		3. Live Testing/: Contains real-world photographs and video demonstrations showing the functional testing and verification of the printed parts on the Nipro machine.

		4. OEM-Reference/: Contains the official Nipro website data, technical specification sheets, and stock machine photos used as a reference during the reverse engineering process.
